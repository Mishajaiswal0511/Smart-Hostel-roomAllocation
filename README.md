**Anti-Gravity** is a futuristic, high-performance web platform designed to revolutionize the hostel experience. By combining advanced Machine Learning algorithms with a premium "Anti-Gravity" glassmorphic UI, it ensures that every student finds their "Perfect Orbit" (the ideal roommate).

---

## ✨ Key Features

### 🤖 1. AI Roommate Advisor
Integrated with a sophisticated chat interface (designed for **Google Gemini API**), the Advisor helps students understand their matches, provides personality insights, and answers questions about hostel life in real-time.

### 🧠 2. Hybrid Matching Engine
Our proprietary matching logic uses a dual-engine approach:
- **K-Nearest Neighbors (KNN):** To find students with similar behavioral profiles.
- **Logistic Regression Model:** Trained to predict compatibility scores based on historical data and weighted preferences.
- **Multi-Factor Scoring:** Considers sleep schedules, study habits, introversion/extroversion, and cleanliness levels.

### 🍱 3. Premium Glassmorphic Dashboard
Inspired by **Apple Vision Pro** and **Iron Man's HUD**, the UI features:
- **Floating Elements:** With soft shadows and neon accents.
- **3D Parallax Effects:** Cards that respond to mouse movement for an immersive feel.
- **Dynamic Backgrounds:** Gaussian-blurred blobs that change state based on user interaction.

### 📊 4. Admin Intelligence Hub
A centralized command center for hostel administrators to:
- Monitor system-wide compatibility averages.
- Manage student records (CRUD operations).
- View real-time matching analytics.

## 🚀 Getting Started

### 📋 Prerequisites
- Python 3.8+
- pip (Python package manager)

### ⚙️ Installation

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Mishajaiswal0511/smart-hostel-room-allocation.git
   cd smart-hostel-room-allocation
   ```

2. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Launch the Application:**
   ```bash
   python app.py
   ```

4. **Access the Dashboard:**
   Open your browser and navigate to `http://127.0.0.1:5000`
